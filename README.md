## Install

```bash
npm i -D tsdown
```

## Usage

```bash
npx tsdown
```

## Migrate from tsup

```bash
npx tsdown migrate
```

Please make sure to commit your changes before migrating. For more details, see the [Migration Guide](https://tsdown.dev/guide/migrate-from-tsup).
